How to String a Tennis Racket

  1. Prepare the racket and strings
    1. Remove the old strings
    2. Clean the frame
    3. Measure and cut the new string
  2. Start the mains (vertical strings)
    1. Thread the first two mains
    2. Tie off the knots
    3. Weave the mains through the racket
  3. String the crosses (horizontal strings)
    1. Start weaving the first cross
    2. Alternate the weaving patterns
    3. Finish and tie off the crosses

Code Snippet: Self Join Command

The Self Join SQL command is a regular join but the table is joined with itself.

FROM table1 T1, table1 T2
WHERE condition;
